## 一、具体步骤

### 1、下载源码

```bash
git clone git@github.com:casdoor/casdoor.git
```

### 2、安装 MySQL

下载地址：https://dev.mysql.com/downloads/mysql/

### 3、修改配置

![An image](/images/RD/Casdoor-1.png)

### 4、安装 web 依赖

![An image](/images/RD/Casdoor-3.png)

定位到 web 目录，安装依赖：

```bash
npm install
```

### 5、运行 GO

```bash
go run main.go
```

### 6、运行 web

```bash
npm run start
```
