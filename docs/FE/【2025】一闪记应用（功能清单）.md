# 一闪记应用（功能清单）

## 1. 创建笔记

- 添加了底部固定的保存按钮
- 实现了保存功能和路由跳转逻辑
- 添加了相关样式，确保按钮固定在底部
- 为编辑器添加了底部边距，避免内容被按钮遮挡
- 添加了加载状态和提示信息
- **注意：** 需要根据实际 API 实现 `saveNote` 方法。

## 2. 笔记列表页面

**主要功能：**

### 搜索和筛选功能：

- 支持按内容搜索
- 支持按笔记本、标签、时间筛选

### 笔记列表展示：

- 显示标题、预览内容和标签
- 支持左滑显示编辑和删除按钮

### 标签管理：

- 支持添加和删除标签
- 标签可用于笔记分类

### 编辑功能：

- 点击编辑跳转到编辑器页面
- 传递笔记 ID 用于加载笔记内容

**需要实现的 API：**

- 获取笔记列表
- 删除笔记
- 标签管理相关接口

## 3. 分类与标签

**主要功能：**

### 笔记本管理：

- 添加新笔记本
- 删除现有笔记本（带确认提示）

### 标签管理：

- 添加新标签
- 删除现有标签

### 使用模拟数据展示效果

- 添加了相应的交互提示
- **注意：** 需要根据实际需求对接相应的 API。

## 4. 同步管理页面

**主要功能：**

### 同步状态显示：

### 同步设置：

- 自动同步开关
- 同步周期选择

### 设备管理：

- 显示已同步设备列表
- 设备在线状态显示

### 操作功能：

- 手动同步
- 清除本地缓存

**需要实现的 API：**

- 同步数据接口
- 清除缓存接口
- 设备状态更新接口

## 5. 共享与协作
