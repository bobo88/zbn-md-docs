# 开放 API 中心（前端原型图）

## 1. 登录页面 ✅

**功能：** 用户输入邮箱/用户名、密码，支持 OAuth 登录（如微信、微博等）。

![An image](/images/FE/open-auth-1.png)

## 2. 控制台首页

**功能：** 展示 API 请求统计、令牌列表，允许创建、撤销令牌。

![An image](/images/FE/open-auth-2.png)

## 3. 创建令牌页面

**功能：** 用户可以生成新的 API 令牌，设置权限和过期时间。

![An image](/images/FE/open-auth-3.png)

## 4. 令牌管理页面

**功能：** 展示所有令牌，允许撤销、编辑权限、管理过期时间。

![An image](/images/FE/open-auth-4.png)

## 5. 系统设置页面

**功能：** 修改账户信息，设置 API 请求频率、IP 白名单、二次验证等安全措施。

![An image](/images/FE/open-auth-5.png)

## 6. 教程页面

**功能：** 帮助开发者学习如何使用 API 和令牌，提供代码示例、常见问题解答。

![An image](/images/FE/open-auth-6.png)
