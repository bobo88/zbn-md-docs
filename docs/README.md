#### 办公室打印机使用指南

为优化资源利用、提升工作效率，请遵循以下指南：

1. **设备简介**  
   办公室配备支持 A3 幅面的彩色打印机。打印 A3 文档时，请手动放入 A3 纸张。

2. **节约用纸建议**

   - 优先选择黑白打印模式，除非演示文稿、设计图稿等确需彩色输出。
   - 打印前请仔细校对，避免重复或打印不必要的内容。

3. **特定打印需求**  
   开发人员可根据实际工作需要打印必要的技术资料或文档，同时请遵循节约原则，确保纸张有效利用。

我们希望通过以上措施，共同营造环保、高效的工作环境。感谢您的理解与支持！
